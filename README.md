# VI-Cinema-em-Portugal
2º Trabalho de VI

## 1º Passo
Abrir o terminal e escrever:
npm i

## 2º Passo - instalar SASS
npm i -g sass

## 3º Passo - para fazer o css
### Abrir o terminal e escrever:
sass --watch styles/index.scss styles/index.css

Isto faz com que o css "veja" o sass e vá atualizando quando houver qualquer alteração.

Apartir daqui, editamos só os ficheiros .scss
